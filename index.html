JavaScript is a scripting programming language.JavaScript basically reads the codes from the top and keeps coming down. When we write a code, the computer cannot understand it directly. Since JavaScript is a high level language, it is difficult for computers to understand. When we write a code, JavaScript converts that code into machine language so that the computer can easily understand it. JavaScript is also called Just in Time compound compiling language. JavaScript is a single threaded programming language. JavaScript has a data structure called 'call stack'. So JavaScript can only perform one task at a time. Each browser uses an engine to run JavaScript, such as the Google Chrome v8 engine, Firefox SpiderMonkey, Microsoft Edge Cycle. All in all, the code we write is first analyzed by JavaScript, then converted into machine code, and then the converted machine code is sent to the computer. The computer displays it in front of us through a browser. Each browser uses its own engine to run JavaScript
